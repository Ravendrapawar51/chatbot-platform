# Chatbot Platform (MERN Stack)

A minimal full-stack **Chatbot Platform** that allows users to create accounts, manage agents/projects, define prompts, and interact with a chatbot via a chat interface.

This project is designed with **scalability, security, and extensibility** in mind and follows clean backend and frontend architecture principles.
